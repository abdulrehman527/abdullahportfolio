# abdullahportfolio

Frontend-only portfolio project, ready for one-click deployment on Vercel.

## Local development

- Install dependencies: `npm install`
- Start dev server: `npm start`
- Build production bundle: `npm run build`

## Vercel deployment

- Import this repository in Vercel.
- Keep root as the project directory.
- Deploy (configuration is already handled by `vercel.json`).

Vercel uses:
- Build command: `npm install && npm run build`
- Output directory: `build`
